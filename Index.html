HTML Rules

Rule

Description

How to Apply

<!DOCTYPE html>

Type of document, always 1st line of HTML

<!DOCTYPE html>

<html>

Type of language

<html>

<Title>

Name of the page that will display on the tab

<title> Hello World </title>

<div>

Separates the sections of a website

<div>
<h1> Hello World </h1>
</div>

<em>

To change the style of a font to italica.

<em> Hello World </em>

<br>

To space the text to the next line,
and it doesn't need a closing tag.

<br>

<ol> or <ul>
    Creates a list and it can be:
  1. Ordered list (with numbers 1,2,3..)
  2. Unordered list (with bullet points)

<ol>This list can be:
<li>Ordered</li>
<li>Unordered</li>
</ol>

<img>
    It adds an image
  • src= here you will add the link of the image or the location of the image you want to add.
  • alt= you should add here the description of the image or item.

<img src="https.testimage.com"/>
